# STVInsurance

Плагин страхования предметов для Minecraft серверов.  
Поддержка версий: **1.16.5 — 1.21.9+**

## Возможности

- Страхование предметов от потери при смерти
- Выкуп потерянных предметов за игровую валюту (Vault) или донат-валюту (PlayerPoints)
- Система слотов на основе прав (донат-привилегии)
- PvP-механика: при отказе от выкупа предмет передаётся убийце
- Таймер на выкуп с автоматическим истечением
- Полностью настраиваемые GUI-меню, сообщения и звуки
- HEX-цвета в сообщениях (`&#RRGGBB`)

## Установка

1. Скачайте `.jar` файл
2. Поместите в папку `plugins/`
3. Перезагрузите сервер
4. Настройте `config.yml` под свой сервер

## Команды

| Команда | Описание | Право |
|---------|----------|-------|
| `/insurance` | Открыть меню страховщика | `stvinsurance.use` |
| `/insadmin reload` | Перезагрузить конфиг | `stvinsurance.admin` |
| `/insadmin info <игрок>` | Информация об игроке | `stvinsurance.admin` |

**Алиасы:** `/ins`, `/страховка`, `/insadmin`

## Права

| Право | Описание | По умолчанию |
|-------|----------|--------------|
| `stvinsurance.use` | Доступ к системе страхования | Все |
| `stvinsurance.admin` | Админ-команды | OP |
| `stvinsurance.slots.N` | N слотов страхования (2–10) | Нет |

## Система слотов

Количество слотов определяется правами. Без дополнительных прав игрок имеет 1 слот (настраивается в `default-slots`). Выдавая право `stvinsurance.slots.3`, вы даёте игроку 3 слота.

Проверка идёт от максимума к минимуму — применяется наивысшее право.

## Как работает

1. Игрок страхует предмет через GUI-меню (опционально за плату)
2. При смерти застрахованный предмет не выпадает, а попадает в «ожидание выкупа»
3. У игрока есть ограниченное время (по умолчанию 2 часа) на выкуп
4. Выкуп возможен за игровую валюту или донат-валюту
5. Если время истекло или игрок отказался:
   - **PvP-смерть** — предмет передаётся убийце
   - **PvE-смерть** — предмет уничтожается
